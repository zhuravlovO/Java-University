## Логіка роботи

Програма моделює музичний плеєр:
*   **Спадкування:** Є базовий клас `MusicComposition`, від якого походять `Rock`, `Pop`, `Classical`.
*   **Агрегація:** Клас `Album` містить список треків.
*   **Поліморфізм:** Список в альбомі зберігає будь-які типи музики, працюючи з ними універсально.

## Технічна частина

*   **Вхідні дані:** Набір треків різних жанрів з назвою та тривалістю.
*   **Реалізація:** 
    1. Створення об'єкта `Album`.
    2. Додавання спадкоємців `MusicComposition`.
    3. Використання `Collections.sort` для сортування.
    4. Фільтрація списку для пошуку діапазону.
