# Лабораторна робота №4

**Тема:** Відношення між класами. **Залікова книжка:** `14309390`

## Мій варіант

| Параметр | Значення | Опис завдання |
| :--- | :---: | :--- |
| **База** | `Lab 2` | Модифікація завдання другої лаби |
| **Структура** | `Classes` | Text $\rightarrow$ Sentence $\rightarrow$ Word $\rightarrow$ Letter |
| **Дія** | `14` | Видалення повторів першої літери у словах |

## Логіка роботи

Програма розбиває текст на об'єкти:
*   **Text:** Чистить пробіли/табуляції, ділить на речення.
*   **Sentence:** Ділить на слова та розділові знаки.
*   **Word:** Складається з літер (`Letter`). Виконує основну логіку видалення.

## Технічна частина

*   **Вхідні дані:** Рядок з купою зайвих пробілів та табуляцій.
*   **Реалізація:** Клас `Main` з вкладеними класами для кожного елемента тексту.
