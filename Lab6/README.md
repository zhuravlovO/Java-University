## Логіка роботи

Створено власний клас `MusicList`, який імплементує стандартний інтерфейс `java.util.List`.
*   **Внутрішня структура:** Використовує клас `Node`, що має посилання на `next` (наступний) та `prev` (попередній) елементи.
*   **Конструктори:**
    1.  Порожній.
    2.  З одним елементом (MusicComposition).
    3.  Зі стандартної колекції (ArrayList тощо).
*   **Операції:** Реалізовано додавання, видалення, доступ за індексом, ітератор.

## Технічна частина

*   **Реалізація:** "Ручне" керування посиланнями між вузлами списку.
*   **Оптимізація:** Метод `get(index)` шукає елемент з початку або з кінця списку, залежно від того, куди ближче індекс.
