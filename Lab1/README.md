##  Логіка обчислення (C11)
Програма проходить по стовпцях матриці $C$:
*   **Парні стовпці (0, 2...):** Шукає мінімальний елемент (`min`).
*   **Непарні стовпці (1, 3...):** Шукає максимальний елемент (`max`).
*   **Результат:** Сума всіх знайдених `min` та `max`.

## Технічна частина
*   **Вхідні дані:** Випадкові матриця розміром 4x4.
*   **Реалізація:** Клас `Main` генерує матриці, виконує множення та підрахунок суми екстремумів.
*   **Запуск:** Відкрити `src/Main.java` і натиснути `Run`.
